# 🎨 BateriaQ Frontend — UX de Alto Rendimiento

La interfaz de usuario de BateriaQ está diseñada para ser minimalista, rápida y extremadamente útil para el opositor.

## 🚀 Stack Visual
- **Framework**: React 18 + Vite.
- **Estilo**: Vanilla CSS (Variables dinámicas + Glassmorphism).
- **Iconografía**: Emojis nativos y SVG optimizados.
- **Navegación**: React Router v6.

## 💎 Design System
Hemos implementado un sistema de diseño propio basado en componentes atómicos:
- `@components/common/Card.jsx`: Receptáculo estándar con estilos elevados.
- `@components/common/Button.jsx`: Acciones con estados de carga e impacto visual.
- **Accesibilidad**: Temas oscuros por defecto para reducir la fatiga en sesiones nocturnas.

## 🧩 Modos de Juego/Estudio
- **TestPlay**: Interfaz de examen con atajos de teclado (`A`, `B`, `C`, `D`, `Enter`, `M`).
- **AIChatAssistant**: Burbuja flotante de tutoría persistente.
- **Estadísticas**: Visualizaciones de progreso mediante rejillas dinámicas.

## 🛠 Desarrollo
1. `npm install`
2. `npm run dev`
