# 🛠 Manual del Administrador — Centro de Control BateriaQ

Este manual describe las funciones exclusivas para usuarios con rol **ADMIN**.

---

## 📊 1. El Panel de Control (Overview)
Al entrar como administrador, tu vista principal muestra el estado vital del sistema:
- **Métricas Globales**: Total de usuarios activos y volumen de preguntas.
- **Estado del Servicio**: Indica si el microservicio de IA está respondiendo correctamente.

---

## 👥 2. Gestión de Usuarios
Desde la pestaña **"Usuarios"** puedes:
- **Cambiar Roles**: Ascender a un usuario a Administrador o revocar permisos.
- **Eliminar Usuarios**: Acción definitiva para limpiar cuentas inactivas o problemáticas.
- **Monitorizar Oposiciones**: Ver en qué oposición está registrado cada usuario, incluyendo las creadas por ellos mismos.
- **Auditoría de Contenido**: Ten en cuenta que los estudiantes ahora pueden crear sus propias oposiciones y temas. Puedes supervisarlos desde el panel de administración global.

---

## 📚 3. Gestión de Contenido (Temas y Preguntas)
Esta es la parte más crítica para mantener la calidad de la plataforma.

### Pestaña "Temas"
- **Crear Temas**: Define el nombre, descripción e icono (emoji) que verán los alumnos.
- **Dificultad**: Asigna un nivel al tema para que el generador de tests funcione correctamente.

### Pestaña "Preguntas"
- **Listado**: Visualiza el banco de preguntas actual.
- **Filtros**: Puedes buscar preguntas por tema para detectar áreas con poco contenido.
- **Edición**: Próximamente podrás editar el texto de la pregunta, las opciones y la respuesta correcta directamente desde aquí.

---

## 🛡 4. Recomendaciones de Seguridad
1. **Contraseñas**: Recuerda que el sistema exige un nivel alto de seguridad. Nunca compartas tu cuenta de admin.
2. **Logs**: Si detectas lentitud en la IA, el sistema te avisará en la sección de mantenimiento del panel.

---
**Recuerda: Un gran poder conlleva una gran base de datos.**
